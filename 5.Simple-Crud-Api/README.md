## Task
https://github.com/AlreadyBored/nodejs-assignments/blob/main/assignments/crud-api/assignment.md

## Описание

Простое CRUD API на Node.js с использованием in-memory базы данных.

## Установка

- npm install

## Режимы запуска

- npm run start:dev
- npm run start:prod
- npm run start:multi

## API
- `GET /api/users`
- `GET /api/users/:id`
- `POST /api/users`
- `PUT /api/users/:id`
- `DELETE /api/users/:id`
